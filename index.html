    About

    OPAP Draws is nothing more than a Java command line interface application which fetches draws for various games from OPAP. OPAP exposes different draw methods for its game via SOAP, which this application makes use of, or via REST web services.

    The available games are:

    • Kino
    • Lotto
    • Joker
    • Proto
    • Super 3
    • Extra 5

    Also, there are three ways one can fetch a draw:

    • Fetch the latest draw
    • Fetch draws for a specific date
    • Fetch draw for a specific competition number

    One jar to rule them all

    In order to compile OPAP Draws, Apache Maven build tool is required. In root directory, where pom.xml is located, type: 

    mvn package
